# Compress Context Skill

Chay khi conversation qua dai (50+ messages) hoac bat dau session moi.

## Steps

1. **Summarize:** Da lam gi, decisions duoc dua ra, tasks dang lam, blockers
2. **Archive:** Save vao `.claude/memory/archive/YYYY-MM-DD-HH-summary.md`
3. **Rewrite MEMORY.md:** Fresh, concise (duoi 200 lines)
4. **Update decisions.md:** Append any new decisions
5. **Confirm:** "Context compressed. Archive saved to [path]."
